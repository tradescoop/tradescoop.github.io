# SCOOP Token Overview

## Summary
**SCOOP** is the native token of the **Tradescoop Proof-of-Profit Network**, designed to support a transparent social trading ecosystem where verified insights are monetized. The token underpins rewards, access, governance, and deflationary mechanisms within the platform.

---

## Tokenomics

### Core Details
- **Token Name:** Tradescoop by Virtuals  
- **Symbol:** SCOOP  
- **Blockchain:** Base Network  
- **Total Supply:** 1,000,000,000  
- **Initial Distribution:** Fair Launch on virtuals.io  
- **Contract Address:** `0xf404bC113f4fc7C2447CB2556dcF5a56e29fa2dd`

### Launch Platform
- SCOOP is launched on the **virtuals.io** platform.  
- Hosted on the **Base Network**, enabling fast, low-cost transactions.

## Use Cases

### Trader Rewards
- Distributed to traders based on verified performance within the Proof-of-Profit system.

### Payments & Tipping
- Used for payments inside the Tradescoop ecosystem.  
- Enables tipping traders for valuable insights or high-quality ideas.

### Token-Gated Features
- Grants exclusive access to:
  - Premium trading ideas  
  - Market insights  
  - Analytics tools  
  - Data-driven performance dashboards  

### Deflationary Mechanics
- **Token buybacks** reduce circulating supply over time.  

### Governance
- **SCOOP holders** participate in decision-making and influence ecosystem direction.
