# Proof of Profit

## Summary
The **Proof-of-Profit** system ensures every trade on Tradescoop is **verified, transparently tracked, and performance-accurate**. All metrics are calculated automatically in real time, enabling a trustworthy reputation layer for traders.

---

## Performance Tracking

### Automated Verification
- All trades are **automatically tracked** the moment they are opened.
- Trades are verified using platform mechanisms to ensure accuracy and authenticity.
- Data remains **tamper-proof** and reflects actual trading activity.

### Calculated Metrics
Tradescoop computes key performance indicators, including:
- **ROI (Return on Investment)**
- **Win rate**
- **Drawdown**
- **Sharpe ratio**
- **Risk-adjusted returns**
- Additional advanced performance metrics as needed

### Transparency
- All performance data is fully transparent.
- Results cannot be manually edited or manipulated.
- Metrics reflect **real performance**, not promotional claims.

---

## Trader Profiles

### Verified Performance Data
Each trader profile includes:
- **Verified trade history**
- **Risk metrics**
- **Performance summaries**
- **Detailed trade reasoning**

### Follower Tools
Users can:
- Follow top traders  
- Receive **real-time notifications** when new ideas are posted  
- Review transparent, data-driven insights before making their own decisions  

Users remain fully responsible for their own trading actions.

## Leaderboards

### Ranking Logic
Traders are ranked using verified performance metrics. Filters include:
- **Asset class**
- **Risk level**
- **Time period**
- **Strategy type**

### Displayed Statistics
Profiles include:
- Win rate  
- Average return  
- Drawdown  
- Risk-adjusted performance  

Past performance does not guarantee future results.
